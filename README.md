Poker 2
Est une implémentation complète du Texas Hold'em, née de la rencontre entre la tradition séculaire du poker et les possibilités offertes par le code moderne. Ce projet incarne l'ambition de recréer numériquement l'atmosphère d'une vraie table de jeu : la tension des mises, le suspense des cartes retournées, l'intelligence calculatrice des adversaires. Plus qu'un simple jeu, c'est un écosystème où s'entremêlent animations fluides, intelligence artificielle stratégique et architecture réseau pour le multijoueur.

Au cœur de ce projet réside une session de poker complète, orchestrée par une machine à états sophistiquée qui guide le joueur à travers chaque phase rituelle : le mélange hypnotique du paquet, la distribution des cartes dans un ballet animé, les tours d'enchères où chaque décision pèse lourd, puis le dévoilement progressif du tableau commun (flop, turn, river). Le joueur humain affronte trois adversaires contrôlés par des algorithmes qui calculent leurs probabilités de victoire et ajustent leur agressivité en conséquence, capable de bluff comme de prudence calculée.
L'interface répond au moindre mouvement de souris : les cartes s'élèvent légèrement au survol, révélant leur identité en français élégant, les boutons d'action pulsent et changent de couleur, un slider permet de doser précisément chaque mise. Les transitions entre phases ne sont pas de simples coupures mais des effets circulaires qui engloutissent puis révèlent la nouvelle réalité du jeu. Chaque élément graphique a été pensé pour immerger le joueur dans l'ambiance d'un casino virtuel.

Prérequis
Avant de plonger dans l'univers de Poker 2, assurez-vous que votre environnement dispose de Python 3.x ainsi que de la bibliothèque Pygame. Si cette dernière n'est pas encore présente sur votre système, invoquez-la par la formule suivante :

bash

pip install pygame
